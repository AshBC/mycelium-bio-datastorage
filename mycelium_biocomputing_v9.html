Research Project Proposal — 2026

Mycelium
Bio-Computing
& The Farm Server

A comparative study of fungal inventions in data storage,
energy, and regenerative farming compatibility

This project compiles and compares existing mycelium-based biotechnology inventions relevant to biological data storage, with the goal of speculatively designing a mycelium server stack that could operate on a farm or in a climate-controlled growing space. Rather than positioning a single invention as a solution, the project maps the emerging global landscape — cataloguing innovations across universities and companies in the UK, US, Italy, Canada, Australia, and the Netherlands — and evaluates each against three criteria: data storage capability, energy consumption, and compatibility with regenerative farming and CO₂ reduction. The British Mycological Society serves as the primary peer-review and knowledge-dissemination body underpinning this entire field. Drawing on these comparisons, the project works toward a speculative design for a server architecture that is grown, not manufactured — one that could live symbiotically within agricultural infrastructure, consume minimal energy, and decompose harmlessly at the end of its lifecycle.

Generative Speculation — Updated with Database

Mycelium Server Stack

The Speculative Design page reads every entry in the database and selects the best-scoring invention for each functional role — storage, computing, interface, substrate, sensing — using the combined score of all three criteria (data storage + energy + farm compatibility). Each selected invention becomes a labelled component in the stack diagram. Components drawn from real database entries are drawn in orange; components that don't yet have a real invention behind them are drawn in purple with a note that they're speculative or still need to be invented. Dashed lines mark connections between components that haven't been solved yet. As the scraper adds new entries every Friday, the diagram regenerates — if a better-scoring invention appears in any role, it automatically replaces the previous component. The gallery sidebar captures a snapshot every Friday before and after the update so you can see how the design evolves week by week.
